Production

  • Middle Eletronic Box, MEB

       For example, MEB can utilize two VNAs to perform S-parameter measurements across four sites simultaneously, leveraging a algorithm to accelerate test time.

  • Testing Program OEM

    •    Ohmplus is capable of supplying the testing program as an OEM solution and integrating it into mass production through handler or prober integration.

       The test items for RF FEM also include S-parameters, Error Vector Magnitude (EVM), Spectral Emission Mask (SEM),
       Third-order Input Intercept Point (IIP3), Adjacent Channel Leakage Ratio (ACLR), and more.
